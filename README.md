# 🛫 CDG Airport Map - Premium Interactive 3D/2.5D Navigation

A sophisticated, production-ready web application for Paris-Charles de Gaulle Airport (CDG/Roissy) featuring an interactive 3D/2.5D isometric map with real-time flight data, terminal navigation, and passenger services discovery.

## ✨ Features

### 🗺️ Interactive Map
- **3D/2.5D Isometric Visualization**: Beautiful, realistic rendering of all CDG terminals
- **Toggle Modes**: Switch between 3D and flat views seamlessly
- **Clickable Elements**: Terminals (1, 2A-2G, 3), Gates, Points of Interest, Services

### 🏢 Terminals & Details
- **Terminal 1**: Iconic circular architecture
- **Terminal 2A-2D**: Connected terminals with shared services
- **Terminal 2E**: Large long-haul hub with Air France lounges
- **Terminal 2F**: Dual hall structure (F21-F36, F41-F56)
- **Terminal 2G**: Separate satellite with shuttle connectivity
- **Terminal 3**: Compact terminal with transport hub integration

### ✈️ Flight Information
- Live flight status simulation
- Gate assignments
- Departure/arrival times
- Airline information
- Walking time estimates

### 🔍 Search & Discovery
- Global search across all elements
- Category filtering
- Interactive legend
- Quick access favorites

### 📱 Responsive Design
- Desktop-optimized with full 3D experience
- Mobile-friendly navigation
- Touch-gesture support
- Adaptive panels

## 🏗️ Tech Stack
- **Frontend**: React 18 + TypeScript
- **3D**: Three.js + React Three Fiber
- **State**: Zustand
- **Build**: Vite

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Opens at http://localhost:5173
